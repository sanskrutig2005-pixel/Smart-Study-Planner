📚 Smart Study Planner

A simple and responsive web application that helps students organize study sessions, track study and break hours, and maintain a healthy study-life balance.

📸 Project Preview

"Smart Study Planner" (images/screenshot.png)

✨ Features

- Add study sessions with subject names
- Track study hours and break hours
- Automatic calculation of total study hours
- Automatic calculation of total break hours
- Smart Balance Score based on study-break ratio
- Remove unwanted sessions
- Responsive design for desktop and mobile devices

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open "index.html" in your browser.
4. Add study sessions and track your progress.

📊 How It Works

1. Enter a subject/topic name.
2. Enter study hours.
3. Enter break hours.
4. Click Add Session.
5. View updated statistics on the dashboard.
6. Remove sessions when needed.
